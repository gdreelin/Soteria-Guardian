# Soteria Guardian

Soteria Guardian is a Home Assistant-first safety, monitoring, and caregiver visibility platform for elderly residents living independently.

## Core ecosystem
- Cross-platform mobile caregiver application
- Home Assistant control plane
- Aqara FP2 presence monitoring
- Aqara Camera Hub G350 video verification
- Aqara Mini Switch wearable and fixed SOS workflows
- Amazon Echo secondary voice support
- Visual smart-light alerts for hearing-impaired residents
- Future AI event correlation, custom hardware, and reporting

## Guiding principle
No single camera, sensor, wearable, or voice assistant is treated as the only safety mechanism. Soteria Guardian uses layered detection, verification, notification, acknowledgment, and escalation.

## Safety notice
Soteria Guardian is not a medical device, does not guarantee fall detection, and is not a replacement for 911 or a professionally monitored medical-alert service.
